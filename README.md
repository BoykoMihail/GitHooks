# GitHooks

## Установка:
Скопировать файл "prepare-commit-msg" в папку ./git/hooks

### Проверено в следующих GUI-клиентах:
    - Sublime merge
    - Fork
    - Tower
    - SourceTree
    - GitKraken 
    
Для GitKraken необходима команда chmod +x prepare-commit-msg в папке с хуками (.git/hooks)
